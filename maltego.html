⚠️ Legal Warning

Maltego is used for gathering and analyzing public information (OSINT).

Only investigate data that is publicly available or that you have permission to analyze.

Do not use it to harass, track, or target individuals.

What is Maltego?

Maltego is an OSINT (Open Source Intelligence) tool used to collect and visualize relationships between data such as domains, emails, IPs, and people.

What does it do?

  • Maps connections between data points
  • Finds related domains, emails, and IPs
  • Visualizes data in graph form
  • Helps investigations and research

Basic Steps

1. Open Maltego
2. Create a new graph
3. Add an entity (domain, email, etc.)
4. Run "transforms"
5. Analyze the results visually

Common Entities

Domain
IP Address
Email Address
Person
Website

Why it's used

Maltego is used by investigators, security analysts, and researchers to understand relationships between data and uncover useful insights from public information.
