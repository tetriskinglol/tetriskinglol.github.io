Cybersecurity

Field: IT Security

Focus: Protecting data & systems

Key Areas: Networks, Devices, Users

Overview

Cybersecurity is the practice of protecting systems, networks, and data from digital attacks. These attacks often aim to steal, change, or destroy sensitive information.

Common Threats

  • Malware: Harmful software like viruses and ransomware.
  • Phishing: Fake messages used to steal personal data.
  • DDoS Attacks: Overloading servers to shut them down.
  • Password Attacks: Attempts to guess or steal passwords.

Protection Tips

  • Use strong passwords
  • Enable Two-Factor Authentication (2FA)
  • Keep software updated
  • Avoid suspicious links
  • Use antivirus software

Useful Tools

  • Antivirus programs
  • Password managers
  • VPN services

Key Terms

Term Meaning
Encryption Converting data into code for protection
Firewall Blocks unauthorized access
VPN Secure internet connection
Data Breach Exposure of sensitive data
