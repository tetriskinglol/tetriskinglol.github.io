Metasploit is a powerful security testing tool.

Only use it on systems you own or have explicit permission to test.

Unauthorized use may be illegal.

What is Metasploit?

Metasploit is a penetration testing framework used to find and test security vulnerabilities in systems.

What does it do?

  • Find vulnerabilities
  • Run exploit code in a controlled environment
  • Simulate real-world cyber attacks (ethically)

Basic Commands

msfconsole

search exploit

use exploit/example

set RHOST 127.0.0.1

run

Why it's used

Security professionals use Metasploit to test defenses and improve system security before attackers can exploit weaknesses.
