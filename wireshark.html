⚠️ Legal Warning

Wireshark captures network traffic.

Only monitor networks you own or have permission to analyze.

Capturing other people’s data without permission may be illegal.

What is Wireshark?

Wireshark is a network protocol analyzer used to capture and inspect data traveling across a network in real time.

What does it do?

  • Captures live network traffic
  • Analyzes packets in detail
  • Helps troubleshoot network issues
  • Detects suspicious or unusual activity

Basic Steps

1. Open Wireshark
2. Select your network interface (Wi-Fi / Ethernet)
3. Click "Start Capture"
4. Watch packets in real time
5. Use filters to narrow results

Example Filters

http
dns
ip.addr == 192.168.1.1
tcp.port == 80

Why it's used

Wireshark is used by network engineers and security professionals to understand network behavior and identify potential security issues.
