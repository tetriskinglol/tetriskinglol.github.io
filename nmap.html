What is Nmap?

Nmap (Network Mapper) is a widely used open-source tool for network discovery and security auditing. It helps administrators understand what devices are connected to a network and what services they are running.

What it is used for

  • Discovering devices on a network
  • Identifying open ports
  • Checking running services
  • Basic network security auditing

Basic Concept Example

Instead of guessing what is on a network, Nmap maps it out.

Network → Scan → Results (devices + services)

Why it matters in cybersecurity

Ethical hackers and security professionals use Nmap to understand attack surfaces and improve system security before attackers can exploit weaknesses.
