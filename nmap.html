Overview

Nmap (Network Mapper) is a tool used to scan networks and discover devices, open ports, and services running on a system.

Features

  • Host discovery
  • Port scanning
  • Service detection
  • Operating system detection

Basic Usage

Run a simple scan:

nmap 192.168.1.1

Scan a range of ports:

nmap -p 1-1000 192.168.1.1

Examples

Quick scan:

nmap -F 192.168.1.1

Detailed scan:

nmap -A 192.168.1.1
