What is Nmap?

Nmap (Network Mapper) is a tool used to scan networks and discover devices, open ports, and services running on a system.

What it Can Do

  • Detect active devices on a network
  • Find open ports
  • Identify running services
  • Check for potential security risks

Example Command

This is a basic example of how Nmap is used:

nmap 192.168.1.1

This scans a device on your local network.

Important Notes

  • Only scan networks you have permission to test
  • Some scans can be detected by security systems
  • Nmap is used by both security professionals and attackers
